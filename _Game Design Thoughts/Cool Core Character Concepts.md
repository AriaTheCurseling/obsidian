

##### Buff sharing - Support class
Core ability lets the player transfer a buff from themselves to an ally. 

Ideally the most effective gameplay is when both use the buff, ways to encourage:
1. Let the buff get partially refreshed when shared (only one person has it at a time)
2. Duplicate the buff when shared (both players can have the same buff at the same time)

##### "Grap"ling gun
